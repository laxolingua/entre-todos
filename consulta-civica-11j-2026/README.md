# Página acompañante · Consulta Cívica Cubana

Página estática y autónoma preparada para publicarse bajo una ruta como:

`https://entre-todos.org/consulta-civica/`

## Archivos

- `index.html`: contenido y estructura.
- `styles.css`: diseño responsive sin dependencias externas.
- `script.js`: navegación móvil, animaciones y demostración local.
- `assets/insights-11j.png`: evidencia visual del alcance de la publicación.
- `assets/og-consulta-civica.png`: imagen para compartir la página.
- `assets/favicon.svg`: icono del sitio.

## Publicación

Subir el contenido completo de esta carpeta a `/consulta-civica/` dentro del sitio. No requiere compilación, base de datos ni paquetes de JavaScript.

## Demostración

La selección SÍ/NO funciona únicamente en el navegador. No usa formularios, cookies, almacenamiento local, analítica ni solicitudes de red. No registra ni contabiliza la selección.

## Transparencia documental

La versión actual resume la carta y mantiene el texto íntegro fuera de publicación hasta que se confirme su recepción o concluya el plazo inicial de seguimiento. Cuando se decida publicar los PDF, debe añadirse una sección de descargas con ambas versiones firmadas.
